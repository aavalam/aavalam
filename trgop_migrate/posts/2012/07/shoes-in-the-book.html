BTW, the following is my two cents.

1. Example 1-2. Shoes doodler

    button, left, top = self.mouse

  This is okay, but no need to add `self.` in this case.
  You can write like this:

    button, left, top = mouse


2. Example 7-13. Obstacles

    animate(FPS) do
      click do |button, left, top|
        $obstacles << Vector[left,top]
      end
       :
    end

The click block is written within animate block. But it's better to remove the click block from the animate block like this:

    click do |button, left, top|
        $obstacles << Vector[left,top]
    end
    animate(FPS) do
       :
    end

Because the click block is called when user click mouse button automatically.
