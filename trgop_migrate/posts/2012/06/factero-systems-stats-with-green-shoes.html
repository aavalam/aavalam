Factero: Systems stats with Green Shoes.

June 28 2012, 5:44 AM  by ashbb

I'd like to introduce a fantastic Green Shoes app, Factero, created by James.

How do I install on Windows?

Open your console window and do the following steps:

- gem install facter
- gem install clipboard
- gem install ffi        #x86-mingw32
- gem install green_shoes

- git clone git://github.com/jrgifford/factero.git
- cd factero
- gshoes main.rb

That's all. Let's have fun with Shoes! :-D
