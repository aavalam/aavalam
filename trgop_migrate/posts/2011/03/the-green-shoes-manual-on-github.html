The Green Shoes Manual on Github

March 27 2011, 12:50 AM  by ashbb

Finally I can get the Green Shoes HTML manual!

http://ashbb.github.com/green_shoes/

This is created by Green Shoes itself. So, you can also do it by yourself. ;-)

- open built-in manual
- click the `to_html` link
- chose a directory where the html manual will be stored

This is almost the same as Red Shoes HTML manual.
Because I borrowed several code from Red Shoes written by _why.

I think you can create your own manual with Green Shoes. ;-)
