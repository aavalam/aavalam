green_shoes gem is now on RubyGems.org

February 2 2011, 7:07 AM  by ashbb

Ruby-GNOME2 0.90.7 released : "MS-Windows" GTK+ theme by default on Windows.

So, quick! Thanks, Kouhei Sutou.

We can install and run Green Shoes in just 3 steps.
Now, no more additional steps. ;-)

1. download and install rubyinstaller-1.9.2-p136.exe

2. gem install gtk2 --no-ri --no-rdoc

3. gem install green_shoes --no-ri --no-rdoc

Done!
