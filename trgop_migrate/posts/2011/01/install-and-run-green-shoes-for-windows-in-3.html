Install and run Green Shoes for Windows in 3 steps

January 31 2011, 7:27 AM  by ashbb

Ruby-GNOME2 0.90.6 released

This is awesome. Now you can install and run Green Shoes for Windows in just 3 steps.

1. download and install rubyinstaller-1.9.2-p136.exe

    C:\>ruby -v
    ruby 1.9.2p136 (2010-12-25) [i386-mingw32]

2. gem install gtk2 --no-ri --no-rdoc

    C:\>gem install gtk2 --no-ri --no-rdoc
    Successfully installed pkg-config-1.0.8
    Successfully installed glib2-0.90.6-x86-mingw32
    Successfully installed atk-0.90.6-x86-mingw32
    Successfully installed cairo-1.10.0-x86-mingw32
    Successfully installed pango-0.90.6-x86-mingw32
    Successfully installed gdk_pixbuf2-0.90.6-x86-mingw32
    Successfully installed gtk2-0.90.6-x86-mingw32
    7 gems installed
   
    C:\Users\asa>gem list
   
    *** LOCAL GEMS ***
   
    atk (0.90.6 x86-mingw32)
    cairo (1.10.0 x86-mingw32)
    gdk_pixbuf2 (0.90.6 x86-mingw32)
    glib2 (0.90.6 x86-mingw32)
    gtk2 (0.90.6 x86-mingw32)

    pango (0.90.6 x86-mingw32)

    rdoc (2.5.8)

3. git clone git://github.com/ashbb/green_shoes.git

That's it!
Now you can run all samples.

    C:\ashbb\tmp\green_shoes\samples>ruby sample1.rb
