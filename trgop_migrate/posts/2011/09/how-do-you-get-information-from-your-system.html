How do you get information from your system using RVM?
As I use RVM, and by extension, Ruby, there are a few directories that I have a .rvmrc file in that does various things. In one of the root directories of my system, I like to know some additional information the first time I come in.

In that directory, my .rvmrc file looks like this:


rvm use ruby-1.9.2-p290@RubyLearning
erb ~/MOTD

And my erb file looks like this:


* Welcome to <%= `/bin/hostname -s`.strip %> *
There are <%= `/usr/bin/who`.split("\n").size %> active sessions as of <%= Time.now %>.
Remember - nothing unreal exists so if you see Santa in here, give him a wave.

Of course, the quote is just something fun. But knowing how many active sessions is open is a 'good thing to know'(tm).
