gshoes -p

December 24 2011, 11:12 PM  by ashbb

Green Shoes v1.1 has a gem executable command `gshoes`.

C:\tmp>gshoes -h
Usage: gshoes (options or app.rb or app.gsy)
  -m, -men     Open the built-in English manual.
  -mjp         Open the built-in Japanese manual.
  -p           Package a Green Shoes app as a .gsy file.
  -v           Display the version info.
  -h           Show this message.

This is a tiny demo that I tried to make a shoeWax.gsy file.

In the above demo, I can launch a Green Shoes app "shoeWax" by double-clicking, because of editing the Windows registry file on my Windows 7 by using the Microsoft Registry Editor (regedit.exe).

- HKEY_CLASSES_ROOT
  - gsy_auto_file
    - DefaultIcon  ===> C:\ruby\lib\ruby\gems\1.9.1\gems\green_shoes-1.1.348\bin\gshoes.ico,0
    - shell
      - open
        - command  ===> "C:\ruby\lib\ruby\gems\1.9.1\gems\green_shoes-1.1.348\bingshoes.bat" "%1"

Have fun with Green Shoes. :)
