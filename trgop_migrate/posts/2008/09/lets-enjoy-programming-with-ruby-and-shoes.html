Hi,
I'm Satoshi Asakawa, online nickname is ashbb, a Japanese Ruby and Shoes enthusiast. And now an Asst. Teacher of the RubyLearning.org with this blog master Vic. He invited me to write a blog about Shoes a few days ago. So, I'm here. :)

I'm now writing Shoes Tutorial as the text of POIRPC101. If you are interesting about Ruby and Shoes programming, you can enroll in the course.

Okay, this is one of the tiny Shoes sample apps written for the last page of the tutorial today. :-D

# sample93.rbShoes.app :width => 400, :height => 75,                        :title => 'Lovely Creaturs' do background "#D0A".."#F90", :angle => 90 x = 0 creatures = %w(loogink yar cy kamome shaha).collect do |c|   image Dir.pwd + "/#{c}.png", :left => x += 60 end                   messages =<<-EOSThx for reading. :)See you!Enjoy Ruby and Shoes!EOS messages = messages.to_a msg = subtitle '', :top => 30, :stroke => white animate(3) do   creatures.each{|c| c.move c.left, rand(15)} end creatures.each do |c|   c.hover{msg.text = strong messages[rand(messages.length)]}   c.leave{msg.text = ''} endend
