Easy and light: Code Highlighter

November 9 2008, 5:06 AM  by Victor Goff
Shoes-0.r1057 uses Code Highlighter to create Shoes Manual into local your pc.

shoes --manual-html

I took a look around. ;-)

Download the zip file. It is included the following.

- Javascript: code_highlighter.js, code_highlighter_ruby.js
- html script: style.css
- sample.html is below.

# sample.rbdef helloputs 'Hello world!'endhello

Run the following Ruby snippet: mkhtml.rb
# mkhtml.rbcode = IO.read('sample.rb')style_css = IO.read('style.css')open('sample.html', 'w') do |f|f.puts style_cssf.print '<pre><code class="ruby">'f.puts codef.puts '</code></pre>'end

The sample.html will be created.
Open the file with your browser. You can get code highlighted web page.

So easy, isn't it? :-D
